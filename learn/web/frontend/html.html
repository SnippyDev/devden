DevDen Learn: HTML

Lesson 1: What is HTML?

  • HTML stands for HyperText Markup Language.
  • It's the standard markup language for creating web pages.
  • HTML describes the structure of a webpage using elements (tags).
  • An HTML document has two main parts: <head> and <body>.

Example:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>

Lesson 2: Basic Tags

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <br>: Line break
  • <strong>: Bold
  • <em>: Italic

Example:

<h1>Main Title</h1>
<p>This is a paragraph.</p>
<strong>Important text</strong>
<em>Emphasized text</em>

Lesson 3: Links & Images

  • <a href="URL">: Hyperlink
  • <img src="image.jpg" alt="description">: Image

Example:

<a href="https://www.google.com">Go to Google</a>
<img src="cat.jpg" alt="A cute cat">

Lesson 4: Lists and Tables

  • <ul>: Unordered list
  • <ol>: Ordered list
  • <li>: List item
  • <table>: Table
  • <tr>: Table row
  • <td>: Table cell
  • <th>: Table header

Example (List):

<ul>
  <li>Apples</li>
  <li>Oranges</li>
</ul>

Example (Table):

<table>
  <tr>
    <th>Name</th><th>Age</th>
  </tr>
  <tr>
    <td>Alice</td><td>30</td>
  </tr>
</table>

Lesson 5: Forms

  • <form>: Creates a form
  • <input>: Input field
  • <label>: Describes input
  • <textarea>: Multiline input
  • <button>: Button
  • <select> + <option>: Dropdown

Example:

<form>
  <label for="name">Name:</label>
  <input type="text" id="name">
  <textarea placeholder="Your message"></textarea>
  <button type="submit">Send</button>
</form>

Lesson 6: Semantic HTML

Semantic elements clearly describe their meaning both to the browser and the developer.

  • <header>
  • <footer>
  • <article>
  • <section>
  • <nav>
  • <main>
  • <aside>

Example:

<article>
  <h2>Blog Post</h2>
  <p>Content goes here...</p>
</article>

Lesson 7: Multimedia & Embeds

  • <video> and <audio>: Embed media
  • <iframe>: Embed external content (e.g., YouTube)

Example (Video):

<video controls>
  <source src="video.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

Example (YouTube Embed):

<iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID" frameborder="0" allowfullscreen></iframe>

Lesson 8: HTML5 & Best Practices

  • Use semantic tags for better structure and accessibility.
  • Always close your tags properly.
  • Use `alt` attributes on images for screen readers.
  • Validate your HTML with tools like W3C Validator.
  • Use indentation and comments for clean code.

Example (HTML5 Document):

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML5 Page</title>
  </head>
  <body>
    <header><h1>Welcome</h1></header>
    <main>
      <article>Content here</article>
    </main>
    <footer>© 2025</footer>
  </body>
</html>
